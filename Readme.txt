We made Resume Builder app. In that we are using SQLite Database. We are using following features 
RecycleView
Bitmap and BitmapFactory for storing image in databae
itext library for making pdf of resume
Bitmap
Environment
android.webkit.MimeTypeMap
java.io.* library

App Description

In our app user should add his/her details about his/her personal info, study info,his/her projects and other info which are needed to make resume. He/she can upload his/her photo and signature. Then generate pdf of resume which are stored in Resume folder which are created when app is downloaded.
