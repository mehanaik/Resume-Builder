This is a Resume Builder app. Here SQLite Database is used. The following features have been used here: 
RecycleView
Bitmap and BitmapFactory for storing image in databae
itext library for making pdf of resume
Bitmap
Environment
android.webkit.MimeTypeMap
java.io.* library

App Description

In this app user is able to add his/her details about his/her personal info, study info,his/her projects and other info which are needed to make resume. He/she can upload his/her photo and signature. Then generate pdf of resume which are stored in Resume folder which are created when app is downloaded.
